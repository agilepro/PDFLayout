PDF Layout

This is a library for doing basic word-processor style layout in order to create good PDF documents.

Please find the JavaDoc for the PDF Layout at http://purplehillsbooks.com/dist/pdflayout/javadoc/

  • pdflayout-1.0.jar - April 29, 2022

Find more information at: PDFLayout on GitHub.

Release notes

April, 2022

  • PDFDoc object represents the main document, which contains Frames, which can themselves contain more Frames and ultimately Paragraphs that contain text.
  • Fully page width frames supported
  • Addressed some bugs around empty paragraph objects
  • added programmable space before and after paragraphs

the first release. This library is based on pdfbox-layout by Ralf Stuckert and his implementation of the basic frames, paragraphs, and the wordwrap and layout have worked essentially flawlessly. But I had to fork a new version for a couple of reasons: I needed space before and after paragraphs, I need to implement bullet points, and I needed frames that would extend across the page. The code I was integrating this into worked in a way that was not quite compatible to the interface provided. Since then, I have extended the library extensively, but of course, make all the source freely available on GitHub.
