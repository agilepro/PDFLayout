PDF Layout

This is a library for doing basic word-processor style layout in order to create good PDF documents.

Please find the JavaDoc for the PDF Layout at http://purplehillsbooks.com/dist/pdflayout/javadoc/

  • pdflayout-1.0.jar - April 29, 2022

Find more information at: PDFLayout on GitHub.

Release notes

May, 2022

  • Table support: new classes Table, ColumnDefinition, TableRow implemented. Request a table from any Frame object, specify column widths (fixed), then request TableRows as many as needed. Table cells are Frame objects with all the border, padding, and margin capabilities. Entire TableRow must fit on page or it is moved to the following page.

April, 2022

  • Checks for characters that are not in the font, and replaces them with '?' to avoid an exception failure
  • PDFDoc object represents the main document, which contains Frames, which can themselves contain more Frames and ultimately Paragraphs that contain text.
  • Fully page width frames supported
  • Addressed some bugs around empty paragraph objects
  • added programmable space before and after paragraphs
  • Added settings on Frame to allow:
    • keeping an entire frame together
    • forcing a pagebreak before a frame
    • setting a minimum space to reserve at bottom of page, forcing page break if too close to the bottom

the first release. This library is based on pdfbox-layout by Ralf Stuckert and his implementation of the basic frames, paragraphs, and the wordwrap and layout have worked essentially flawlessly. But I had to fork a new version for a couple of reasons: It was not protecting against characters not in the font, I needed space before and after paragraphs, I need to implement bullet points, and I needed frames that would extend across the page, I need ability to force a pagebreak before a frame (without inserting a separate element) and ability to control keep-together on a frame, and keep-with-next minimum space capability. The code I was integrating this into worked in a way that was not quite compatible to the interface provided. Since then, I have extended the library extensively, but of course, make all the source freely available on GitHub.
